# Basiq360: Loyalty Backend

- Tech Stack:
    - Node
    - Express

- Libraries
    - Phone number validation against international formats => libphonenumber-js

- Prisma commands for update 
-npx prisma db pull  - to pull schema from latest DB
-npx prisma generate - to update schema     

-Command for toggle ENV
npm run start -- --env=production
npm run dev -- --env=development


Backend configurations we are using:
express, prisma, node

node version: 22.15.29
database: mysql
server provider: hostdime
process manager on server: pm2
Terminal Multiplexer: tmux
s3 bucket: cloudflare


fontend configuration we are using:
react.js version: 19.1.8
React-native version: 18.2.6
